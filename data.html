Database 1

Database 1 explores the foundational concepts of database systems, with a focus on design, management, and practical application of SQL for data retrieval and analysis. Firm understanding of database structures, relational theory, and the tools needed to effectively work with and manipulate data in real-world scenarios

Concepts

Understanding Databases and Their Impact

  • Impact and importance of database systems today.
  • Describe the function, purpose, and history of databases.
  • Identify database users, components, and primary database models.

Database Design and Components

  • Explain the components of a database system and design database diagrams.
  • Define field names, data types, and primary/foreign keys.
  • Define relationship cardinality and adhere to rules for relational tables.

SQL Querying and Data Retrieval

  • Construct basic SQL queries using SELECT, WHERE, and ORDER BY clauses.
  • Utilize aggregate functions and GROUP BY to summarize data.
  • Develop multi-table SQL queries with INNER and OUTER JOINs.

Advanced SQL Operations

  • Incorporate sub-queries for filtering via WHERE clause.
  • Combine multiple queries using UNION.
  • Work with multiple tables to aggregate and calculate new data efficiently.
  • Relational Model
  • Multi-Table Outer Joins
  • Case Study Script

Database 2

Database 2 delves into the evolution of databases and DBMS, covering key concepts, technologies, and practical skills. Learn to design, implement, and optimize relational databases using SQL, while exploring advanced topics like normalization and performance tuning.

Concepts

Database History & Terminology

  • Understand the evolution of databases and DBMS.
  • Identify key database terminology and concepts.
  • Explore the components and architecture of a database system.

Relational Database Design

  • Model data using Entity-Relationship Diagrams (ERDs).
  • Normalize database designs to improve data integrity.
  • Implement relational databases using SQL, including query writing, data manipulation, and script execution.

Database Management and Administration

  • Install, configure, and manage database systems.
  • Secure databases by managing user access and permissions.
  • Optimize database performance through query optimization and index tuning.

Advanced Database Topics

  • Analyze business requirements to identify data needs.
  • Investigate current and future trends in database technology.
  • Compare different database technologies and their features.
  • Database Creation Script
  • Tableau Sales Data

Introductory ETL

Introductory ETL brings the ETL (Extract, Transform, and Load) process to life focusing on the integration, transformation, and loading of data. Learn to handle multiple data sources to support business intelligence analysis and experience industry-standard tools, best practices, and security measures to build efficient workflows.

Concepts

Data Integration and ETL Basics

  • Define ETL (Extract, Transform, Load) and its importance.
  • Identify data sources and target destinations.

Data Extraction and Transformation

  • Extract data from structured and unstructured data sources, analyzing for dataset labels and content.
  • Apply data validation techniques to maintain data integrity.

Data Loading and Warehousing

  • Load transformed data into target destinations.
  • Determine appropriate load methods based on data requirements.

Data Analysis and Interpretation

  • Interpret data to extract meaningful insights.
  • Assess data usability and determine if it meets business requirements.
  • ETL Case Study
  • Tableau Sales Data

Data Privacy & Security

Data Privacy & Security brings critical aspects of data security and privacy regulations to the forefront with GDPR, HIPAA, and CCPA. Deploy security measures to protect sensitive information with data classification, access control, encryption, and social engineering mitigation.

Concepts

Data Privacy and Regulatory Compliance

  • Explain key data privacy concepts and terminology.
  • Analyze compliance requirements for GDPR, HIPAA, and CCPA.

Securing Sensitive Data

  • Define, classify and protect sensitive data, including PII.
  • Secure infrastructure and databases using best practices.

Social Engineering Defense and Network Security

  • Recognize and mitigate social engineering attacks like phishing.
  • Understand key network security features, vulnerabilities, and best practices for password management.

Incident Response and Emerging Trends

  • Prepare for security incidents by developing response plans, identifying breach types, and preparing incident notification letters.
  • Explore emerging trends in data security and privacy, including AI and ML applications.
